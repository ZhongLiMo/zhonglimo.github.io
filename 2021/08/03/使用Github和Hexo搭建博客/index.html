    使用Github和Hexo搭建博客

    一、准备工作

    注意：本教程适用于windows，没有在linux试过，但是参考的原教程是mac的，理论上差不多。

    1. 注册Github账号：https://github.com/
    2. 安装Git命令行：https://git-scm.com/downloads
    3. 安装node.js：https://nodejs.org/zh-cn/
    4. 安装Hexo：https://hexo.io/zh-cn/
      这一步只要执行npm install hexo-cli -g即可。

    二、搭建博客

    找个目录打开Git命令行，然后执行hexo init Blog(这里Blog是你的博客根目录名，可以自己取)，然后cd Blog，执行npm install，本地博客就搭建完了，就这么简单。

    执行hexo server运行本地博客：

    然后就可以在自己本地浏览器输入http://localhost:4000看到本地博客的效果了。

    三、部署博客

    找到_config.yml配置文件，这里又一些主要的参数信息：

    1. 博客相关信息

    2. 关联Github部署信息

      这里需要注意的是仓库名字要配置成your_github_name.github.com，your_github_name是你的Github账户名，分支自己选吧，我就用了默认的master分支。

    3. 部署到Github
      可以先本地试一下hexo server前面改的标题之类的有没有生效，应该是没问题的。
      然后执行npm install hexo-deployer-git --save，然后hexo deploy(这个是部署命令以后每次修改都要记得重新部署)，第一次执行hexo deploy的时候应该要验证下Github的账户信息，按提示操作即可。
      都弄完了就可以在网页输入your_github_name.github.io看到你的博客了。

    至此，博客搭建结束。

    四、修改主题

    先去https://hexo.io/themes/#clean找到自己喜欢的主题，这个是Hexo官网的一些主题，当然也可以自己做或者自己找一些别的主题。
    找到后一般都有文档介绍，比如我就在这个网址找到第一个主题keep，安装教程。
    安装步骤很简单，执行npm install hexo-theme-keep，然后将_config.yml配置文件中的theme设置为keep。

    然后hexo d重新部署，即可看到新主题界面。

    最后注意：别忘了把这个部署博客的整个目录也存起来，我也是放到了Git上做了个私有仓库，博客的话是公有仓库。

    六、使用hexo-admin插件管理文章

    只需要一条命令npm install hexo-admin --save，然后浏览器输入 http://localhost:4000/admin 即可开始使用了。
